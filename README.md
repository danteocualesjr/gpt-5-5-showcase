# ChatGPT 5.5 Showcase

A polished Vite + React single-page website concept that presents ChatGPT 5.5 through a clean SaaS landing-page design.

## What It Includes

- Premium hero section with product-style positioning.
- Capability cards for reasoning, coding, creativity, research, multimodal thinking, and automation.
- Visual workflow and metrics panels that make GPT's capabilities feel tangible.
- Responsive layout, refined gradients, glass cards, hover states, and subtle animation.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
